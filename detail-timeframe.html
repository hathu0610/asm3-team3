Project Description/Timeframe

  • Week 1 Choose an idea for the project
    Research about IT technologies. Evaluate the pros and cons between each ideas
  • Week 2 Preparation for the project
    Choose the tools and technologies to develop the chatbot. Preparing set of questions (data) to train the chatbot. Setting the frame for the final reports
  • Week 3 Develop the project
    Refine the data to train the chatbot. Creating the website to integrate the chatbot. Writing the final report. Preparing for the presentation
  • Week 4 Finalize the idea for the project
    Writing the final report. Presenting the project
  • Week 5 Finish the first version of the project
    Finishing building the chatbot. Integrating the chatbot to website. Testing the first version and collecting customers' and investors' feedback
  • Week 6 Plan the next phase of the project
    Analyzing the user feedbacks. Researching and comparing between chatbot products’ features. Evaluating the scope and potential development for the project
  • Week 7 Building features: Setting reminders
    Defining the features and write documentation. Designing user flow and UI for the features. Developing the setting reminders flow on the chatbot. Synchronizing the users' reminders with their calendar through Outlook calendar API
  • Week 8 Building features: Check room ability
    Defining the features and write documentation. Designing user flow and UI for the features. Developing the check room ability flow on the chatbot. Requesting for RMIT's room availability's data.
  • Week 9 Building features: Fun facts
    Defining the features and write documentation. Designing user flow and UI for the features. Developing the fun fact flow on the chatbot. Preparing set of fun facts (data) to train the chatbot
  • Week 10 Finish the second version of the project
    Debugging and finalizing the second version of the project. Retrieving customers feedbacks for the second version
  • Week 11 Plan the next phase of the project
    Defining the potential business strategy for the project Analyzing the user feedbacks. Evaluating the scope and potential technical development for the project
  • Week 12 Expand the target customers of the project
    Creating a business model for the project. Finding customers (universities) that interested in trying the product. Researching and scaling the software architect
  • Week 13 Customize our project to customer #1
    Defining the customers’ needs and managing deadlines. Preparing the chatbot’s set of questions. Retrain the model for the customers' chatbot
  • Week 14 Customize our project to customer #2
    Defining the customers’ needs and managing deadlines. Preparing the chatbot’s set of questions. Retrain the model for the customers' chatbot
  • Week 15 Customize our project to customer #3
    Defining the customers’ needs and managing deadlines. Preparing the chatbot’s set of questions. Retrain the model for the customers' chatbot
  • Week 16 Plan the next phase of the project
    Re-evaluating the business model and team's current progress. Creating project/product roadmap for 1 year
